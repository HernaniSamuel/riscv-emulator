pub fn set_pc(&mut self, value: u32) -> Result<(), VMError>

Sets the program counter to value.

The value must point to a valid address in RAM. This method does not enforce instruction alignment; that responsibility belongs to the CPU.

§Errors

Returns VMError::MemoryOutOfBounds if value is outside RAM.
